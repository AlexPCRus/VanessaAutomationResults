Заголовок: Невозможность ввода количества для услуг в документе Заказ

Критичность: Критический

Приоритет: Высокий

Статус: Новый

Окружение: Windows Pro 10, 1С версия 8.3.21.1709, конфигурация "Демонстрационная конфигурация", Файловая база.

Детальное описание:
В программе нет возможности ввести количество для товаров услуг в табличной части "Товары" в новых и существующих документах "Заказ"

Шаги для воспроизведения:
1. открыть список заказов
2. создать новый документ заказ
3. добавить новую строку в ТЧ Товары
4. выбрать товар "Доставка" в поле "Товар" строки ТЧ
5. попытаться ввести данные в поле "Количество" строки ТЧ

Фактический результат: Невозможно ввести данные в поле количество для товаров услуг в документе Заказ

Ожидаемый результат: При вводе данных в поле количество для товаров услуг в документе Заказ они вводятся, происходит расчет суммы по строке и общих количества и суммы заказа

Приложения: Отсутствуют
